Description

Tiles Apart is a single player puzzle game. Slide rows of tiles to make your character traverse the many levels. In this game, you don't control the character; you control the tiles. Slide a row to the left and the character will move to the left. Slide a row to the right and the character will also move to the right.

I worked 4 working days a week, for 8 weeks. I was responsible for creating various Unity Editor tools and level streaming/management.

My tasks

Tools
Tiles

I made a tool for the Unity Editor, to allow designers to easily change individiual tiles on the grid. This tool also allowed the in-editor movement of individual tiles, and was used to create puzzles.

Level streaming
Level Transitions

Using Unity Async functions, I created a system that allowed seamless level transitions. The next level was loaded in on current level completion, and tiles were swapped using a 'sinking' and 'rising' animation, to hide any discrepancies. The end tile with the player on it is transformed to a start tile, and lerped to the new position.
